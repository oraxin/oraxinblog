windows下pip包管理工具的安装

Posted on November 27, 2015

windows下的python2.7默认不带pip包管理工具，要更方便的安装所需要的包，最好还是安装pip这个包管理工具。虽然现在python3.4已经自带pip，单由于还有部分重要包仍旧不支持python3.0+，这还是要自己动手了。

1、在以下地址下载最新的PIP安装文件：https://bootstrap.pypa.io/get-pip.py

2、把文件get-pip.py解压并放到python根目录内

3、在cmd中运行python get-pip.py install，会自动下载和安装

4、添加pip目录到环境变量path中，如C:\python27\Scripts

好了就这么简单。

清迈散心手记一

Posted on October 24, 2015

从清迈回来似乎是很久之前的事情，总想要写点什么，但又常常不知道怎么开头。今天终于耐下兴致，写点流水，谈点回忆。

去清迈，特别是只去清迈，那时七月份做出来的决定。但由于工作，一直没有时间，到了八月底才勉强决定国庆期间出发。然后也把原来的七天缩短到了五天，取消了去Pai县的行程。然后就匆匆的在淘宝上定了机票。

这次机票预订的比较晚了只提前了一个多月，价格不算最便宜来回含税2500多，但好在是东航的航班，不需要花钱买餐食和托运行李了。事实上东航从上海浦东去清迈的航班根本就坐不满，来回都只坐了一半左右的人。我想说的是东航太抠门，为什么不再降点价，而且东航的航班都要坐摆渡车。算了考虑到是国庆期间2500也不算贵了。

然后是订酒店。好多攻略上都说只订第一晚的就好了，清迈到处都是酒店。但还不是考虑到国庆，加上一共才五天，如果去掉来回就四天的活动时间，我可不愿意拉着行李到处找酒店，但在tripadvisor找了好久，好多喜欢的旅店都被订完了，最后选择了big tree boutique hotel。这个酒店还不错，比较新，很干净，房间看出去的风景也挺好，除了晚上外面的车来车往比较吵，还有大多数双条车和tutu车司机不认识，需要准备泰语的地址。我是在agoda上预订的这个酒店，豪华大床房含税400多一晚，真的是不便宜，看样子还是订的太晚还是在节假日啊。

关于参加本地团，这两个团都是在淘宝上的汉清迈预订了，一个是一天的大象训练营，我建议大家选这样的大象营，而不是美沙这类的带有表演项目。这个活动你能学习给大象的命令，虽然大象根本就不听你的，和大象亲密接触，无座椅的直接骑大象，给大象洗澡，整个过程没有任何虐待大象行为。就像我们骑的大象很贪吃，经常就那么不走了就开始吃还不停，象夫只能不断给她发命令，大象不理他他也没办法只能等着。还有一个是清莱的黑庙白庙的一天行程，这个报团是因为清莱的白庙是必看的，而且从清迈到清莱还是比较远，参个团还是比较方便的。

Git常用命令整理

Posted on April 10, 2015

Git是个好东西，虽然我才刚刚使用，但觉得很不错，虽然与svn相比需要使用命令行来操作，对命令还有点记忆障碍。人老了记点东西不容易啊。所以说，我还是要把Git使用中最基本的记录下来。

Git，说实话，我还不怎么会用，如果要系统的学习基础使用，可以参考廖雪峰的Git教程写的比较详细，而我呢，只是把我配置完之后经常需要使用的命令做一个整理。

创建版本库

要开始使用当然要新建版本库，这个很简单，你要在你的空的地方创建一个空目录（为了防止出现奇怪的问题，目录中不要使用中文字符），然后在Git bash等Git命令行中进入此目录，运行

$ git init

这样就把你的目录初始化为版本库了，你会发现当前目录下多了一个.git的目录，这个目录是Git来跟踪管理版本库的，没事千万不要手动修改这个目录里面的文件，不然改乱了，就把Git仓库给破坏了。

开始使用网络访问黑科技

Posted on February 22, 2015

访问国外网站的黑科技是什么。那就是EurekaVPT的SSLedge。

SSLedge 就是 SSL proxy，可以无痛用在 Chrome 上，当然也可以用工具转换成高性能的 HTTP 代理，然后自己配置 Pac 或者其它手段。

SSL proxy 的好处在于无连接，无需每次拨号使用，也比较适合在网络环境恶劣的情况下使用；标准 SSL 加密目前可暂避 SSH 类的封锁；SSL 软+硬加速可提供高性能安全代理；在不安全的环境下，SSL proxy 可充当 SSL webvpn，全局走代理可以提供信息安全。

最近我在EurekaVPT上购买了它的SSLedge一年基础服务，每个月10G流量。实际在使用过程中发现无论开网页还是youtube都是非常的顺畅，相比我原来购买的其他VPN来说，速度好的不是一点两点。虽然如果重度使用的话也许10G流量不够用，但每天更新的流量统计比较保守合理，远远小于你实际使用的用量。还是很人性化的。

虽然我一直都挺喜欢皮卡，对于我这样又矮又瘦的人来说是非常不适合的，喜欢上了坦途。

坦途十几年前丰田品牌通过坦途车型，成功进入了北美中高档皮卡市场，并将对手直接锁定福特F150和道奇公羊车型。

2014款丰田坦途在车款选择上较为为丰富，提供了SR、SR5、Limited、Platinum和1794五个版本；而在车厢结构方面，则分为标准双门版、四门双排座和全尺寸的四门CrewMax版。
